There is / There are

「〜がある／いる」を表す表現です。

What does “There is / There are” mean?

We use “There is / There are” to say something exists or is present.

「There is / There are」は、物や人の存在を表します。

Form

Use “There is” for singular nouns. Use “There are” for plural nouns.

単数名詞には「There is」、複数名詞には「There are」を使います。

Form Use Example
There is Singular noun There is a cat.
There are Plural noun There are two cats.

Examples

There is a book on the table.
日本語を見る / 隠す テーブルの上に本があります。

There are many students in the classroom.
日本語を見る / 隠す 教室にはたくさんの生徒がいます。

There is a dog in the garden.
日本語を見る / 隠す 庭に犬がいます。

Negative Form

Use “There is not” or “There are not.”

否定文は「There is not / There are not」を使います。

Positive Negative
There is a cat. There is not a cat.
There are books. There are not books.

Questions

Put “Is there / Are there” at the beginning.

疑問文は「Is there / Are there」を文頭に置きます。

Question Meaning
Is there a pen? ペンはありますか。
Are there any apples? りんごはありますか。

Exercises

1. Complete the sentences

There is / There are を入れましょう。

a) ___ a cat on the chair. 答えを見る There is

b) ___ three books on the desk. 答えを見る There are

c) ___ a park near my house. 答えを見る There is

2. Translate into English

英語にしましょう。

a) テーブルの上にりんごがあります。 答えを見る There is an apple on the table.

b) 公園に子どもがたくさんいます。 答えを見る There are many children in the park.

Mini‑Quiz

1) 「机の上に本があります。」

a) There are a book on the desk.
b) There is a book on the desk.
答えを見る b) There is a book on the desk.

2) 「箱の中に3つのりんごがあります。」

a) There are three apples in the box.
b) There is three apples in the box.
答えを見る a) There are three apples in the box.
