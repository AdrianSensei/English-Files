Conditionals – Zero, First, Second & Third

（条件文：ゼロ・ファースト・セカンド・サード条件）

Zero Conditional

（ゼロ条件：いつも起こること・事実）

Form:
If + present simple, present simple
（If + 現在形, 現在形）

Use:
We use the Zero Conditional to talk about facts, rules, or things that are always true.
（事実・ルール・いつも起こることに使う）

Examples:

• If you heat water, it boils.
（お湯を温めると、沸騰する。）

• If people don’t eat, they get hungry.
（人は食べないと、お腹がすく。）

First Conditional

（ファースト条件：現実的な未来の可能性）

Form:
If + present simple, will + verb
（If + 現在形, will + 動詞の原形）

Use:
We use the First Conditional to talk about real possibilities in the future.
（未来に起こる“かもしれない”現実的な可能性に使う）

Examples:

• If I study, I will pass the test.
（勉強すれば、テストに合格する。）

• If it rains tomorrow, we will stay at home.
（明日雨なら、家にいる。）

Second Conditional

（セカンド条件：今・未来の「もし〜なら」現実ではない仮定）

Form:
If + past simple, would + verb
（If + 過去形, would + 動詞の原形）

Use:
We use the Second Conditional to talk about unreal or very unlikely situations now or in the future.
（今または未来の「本当ではない／可能性が低い」仮定に使う）

Examples:

• If I had a lot of money, I would travel around the world.
（たくさんお金があれば、世界中を旅行するのに。）

• If she were taller, she would play basketball.
（彼女がもっと背が高ければ、バスケをするのに。）

Third Conditional

（サード条件：過去の「もし〜していたら」実際とは違う仮定）

Form:
If + had + past participle, would have + past participle
（If + had + 過去分詞, would have + 過去分詞）

Use:
We use the Third Conditional to talk about unreal situations in the past and their imaginary results.
（過去に実際は起こらなかったことを想像して話すときに使う）

Examples:

• If I had studied harder, I would have passed the exam.
（もっと勉強していれば、試験に合格していたのに。）

• If they had left earlier, they would have caught the train.
（もっと早く出ていれば、電車に間に合っていたのに。）

Exercises

（練習問題）

Zero Conditional

（ゼロ条件）

  1. If you touch fire, it ______ you.
    （火に触ると、〜する。）
    Answer burns you
    （やけどする）
  2. If students are late, the teacher ______ angry.
    （生徒が遅刻すると、先生は〜。）
    Answer gets angry
    （怒る）

First Conditional

（ファースト条件）

  1. If it rains this afternoon, we ______ at the café.
    （今日の午後雨なら、カフェで〜。）
    Answer will stay
    （いる／待つ）
  2. If you hurry, you ______ the bus.
    （急げば、バスに〜。）
    Answer will catch
    （間に合う／乗れる）

Second Conditional

（セカンド条件）

  1. If I lived near the sea, I ______ every day.
    （海の近くに住んでいたら、毎日〜のに。）
    Answer would swim
    （泳ぐのに）
  2. If he had more time, he ______ a new hobby.
    （もっと時間があれば、新しい趣味を〜のに。）
    Answer would start
    （始めるのに）

Third Conditional

（サード条件）

  1. If we had left earlier, we ______ the traffic jam.
    （もっと早く出ていれば、渋滞を〜のに。）
    Answer would have avoided
    （避けられたのに）
  2. If she had listened to me, she ______ that mistake.
    （私の話を聞いていれば、そのミスを〜のに。）
    Answer would have avoided / wouldn’t have made
    （避けられた／しなかったのに）

Mixed practice (all conditionals)

（ミックス練習：すべての条件）

  1. If water reaches 100°C, it ______.
    （水が100度になると、〜。）※ゼロ条件
    Answer boils
    （沸騰する）
  2. If you don’t eat breakfast tomorrow, you ______ hungry at school.
    （明日朝ご飯を食べなければ、学校で〜。）※ファースト条件
    Answer will be
    （お腹がすく）
  3. If I were you, I ______ to the doctor.
    （私があなたなら、医者に〜。）※セカンド条件
    Answer would go
    （行く）
  4. If they had studied more, they ______ the test.
    （もっと勉強していれば、テストに〜のに。）※サード条件
    Answer would have passed
    （合格していたのに）
