Some / Any

「いくつかの」「少しの」を表す語です。

What do “some” and “any” mean?

We use “some” and “any” to talk about an unspecified amount.

「some」と「any」は、はっきりしない量について話すときに使います。

General Rules

Use “some” in positive sentences. Use “any” in negative sentences and questions.

肯定文では「some」、否定文と疑問文では「any」を使います。

Form Use Example
some Positive sentences I have some apples.
any Negative / Questions Do you have any apples?

Examples

I have some water.
日本語を見る / 隠す 私は水を少し持っています。

She doesn’t have any money.
日本語を見る / 隠す 彼女はお金を持っていません。

Do you have any questions?
日本語を見る / 隠す 何か質問はありますか。

Special Case

We can use “some” in questions when offering or requesting.

申し出や依頼の疑問文では「some」を使うことがあります。

Example Meaning
Would you like some tea? 紅茶はいかがですか。
Can I have some water? 水を少しいただけますか。

Exercises

1. Choose some or any

some か any を入れましょう。

a) I have ___ friends. 答えを見る some

b) Do you have ___ questions? 答えを見る any

c) She doesn’t have ___ time. 答えを見る any

2. Translate into English

英語にしましょう。

a) 私はいくつか質問があります。 答えを見る I have some questions.

b) あなたは何か飲み物がありますか。 答えを見る Do you have any drinks?

Mini‑Quiz

1) 「私はお金を少し持っています。」

a) I have some money.
b) I have any money.
答えを見る a) I have some money.

2) 「彼らは食べ物を持っていません。」

a) They don’t have some food.
b) They don’t have any food.
答えを見る b) They don’t have any food.
